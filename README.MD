# Example of redis with PHP

## Download Redis

https://redis.io/

## Installation

git clone https://github.com/Riper3/php-redis-example

cd php-redis-example

## How to use

php test.php

## Configuration files

### Mysql

* db_host
* db_user
* db_pass
* db_table
* db_charset

### Redis

* rd_host
* rd_port
* rd_expire
